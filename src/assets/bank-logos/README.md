# Bank logo overrides

`AccountLogo` шарит эту папку через `import.meta.glob` и подхватывает
любой `.svg`, чей filename совпадает со `slug` из
`src/lib/bankRegistry.ts`.

## Как добавить логотип

1. Получите SVG-файл (например, со страницы Wikipedia или из brand-kit
   банка) — **убедитесь, что у вас есть право на использование**.
2. Назовите файл по slug-у из реестра, например `sber.svg` для Сбера,
   `tbank.svg` для Т-Банка, `kaspi.svg` для Kaspi.
3. Положите в эту папку. Vite автоматически инлайнит ассет в бандл при
   следующей пересборке.

## Что происходит без файла

`AccountLogo` рисует цветной квадратик с монограммой — берёт фирменный
цвет и аббревиатуру из `BANK_REGISTRY`. Уже выглядит «фирменно»
(зелёный для Сбера, жёлтый для Т-Банка, и т.д.), просто это не точный
логотип.

## Добавить новый банк, которого нет в реестре

Откройте `src/lib/bankRegistry.ts` и добавьте запись по образцу.
Минимум — `slug`, `patterns`, `color`, `fg`, `monogram`, `name`.
